## ❔ Descrição:
Este é um site de busca de animes simples e responsivo, desenvolvido apenas com HTML,CSS e JavaScript. 
Escolhi a Kitsu API para fornecer todas as informações possíveis sobre o anime. Você pode tudo sobre esta API clicando <a href="https://kitsu.docs.apiary.io/#">aqui</a>
